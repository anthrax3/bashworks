todo_add()

Adds a todo to the list.

Log

  • Error Usage: $usage
  • Info Added: $todo_name

Param

  • Todo name

Return

  • 2 If the todo name param was not given.

todo_delete()

Removes a todo from the list.

Log

  • Error Usage: $usage
  • Warn #$todo_id: empty name
  • Info Removed #$todo_id: $todo_name

Param

  • Index of the todo.
